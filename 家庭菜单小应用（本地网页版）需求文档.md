# 家庭菜单小应用（本地网页版）需求文档

## 一、产品概述

### 1. 产品目标

为家庭成员（含老人、小孩）提供零门槛、本地运行的菜品管理与菜单规划工具，解决 “每日点餐难” 问题，支持菜品分类、日历规划、智能推荐、网页端跨设备访问，界面卡通友好、操作极简。

### 2. 核心优势



* 本地运行：无需服务器部署，数据存储在本地（浏览器 / 本地文件），无网络也可用

* 跨设备适配：支持电脑、手机、平板通过浏览器访问（同一 WiFi 环境下共享）

* 全员易用：大字体、卡通风格、少操作步骤，老人小孩均可独立使用

## 二、核心功能需求

### 1. 菜品库管理模块

#### 1.1 菜品基础操作



* 新增菜品：提供简洁输入框，支持录入「菜品名称」，选择「分类标签」（默认：肉类、蔬菜类、汤类，支持自定义添加 / 删除分类）

* 编辑 / 删除：菜品列表支持点击修改名称、调整分类，或直接删除

* 菜品展示：以大卡片形式展示，卡片包含菜品名称、分类标签，文字字号≥16px，便于识别

#### 1.2 菜品搜索与导入



* 本地搜索：支持按「菜品名称关键词」「分类」筛选，搜索结果实时展示，按钮清晰易点

* 网络导入：提供 “网上搜菜谱” 入口，支持按「食材」「口味」筛选（如 “鸡肉”“清淡”），搜索结果展示菜品名称 + 简易配图（可选），点击 “添加到我的库” 即可存入本地

#### 1.3 分类管理



* 默认分类：肉类、蔬菜类、汤类

* 自定义分类：支持新增分类（如 “主食类”“小吃类”）、修改分类名称、删除无用分类

* 分类展示：分类标签以大尺寸按钮呈现，点击可快速筛选对应分类菜品

### 2. 日历式菜单规划模块

#### 2.1 日历界面



* 展示形式：按月视图日历展示，日期卡片尺寸放大（手机端点击无压力），当日菜单以简洁文字标注在日期下方

* 切换便捷：支持左右滑动 / 点击箭头切换月份，当前日期高亮显示

#### 2.2 菜单编辑



* 手动添加：点击日期卡片，弹出 “添加菜品” 窗口，可从菜品库勾选 1-N 道菜品（支持荤素搭配提示），确认后保存至当日菜单

* 随机抽取：点击 “随机选菜” 按钮，可设置抽取规则（如 “1 荤 1 素 1 汤”“2 素 1 汤”），从菜品库对应分类中随机生成菜单，支持手动替换其中菜品

* 菜单修改：点击已规划日期的菜品，可删除、替换菜品

#### 2.3 荤素搭配推荐



* 自动推荐：点击 “搭配推荐” 按钮，基于菜品库分类，智能生成荤素均衡的菜单组合（如 “红烧排骨 + 清炒时蔬 + 番茄蛋汤”）

* 推荐调整：支持手动替换推荐中的任意菜品，调整后自动保存

### 3. 提醒功能模块



* 提醒设置：规划当日菜单后，可设置提醒时间（如 “做饭前 1 小时”“上午 10 点”），支持选择提醒频率（仅当日 / 每周重复）

* 提醒方式：通过浏览器通知弹窗提醒（电脑 / 手机端均支持），弹窗文字放大、配色醒目，附带当日菜单简要提示

* 提醒管理：支持查看已设置的提醒列表，可修改提醒时间或删除提醒

### 4. 数据存储与同步模块



* 本地存储：核心数据（菜品库、菜单规划、提醒设置）存储在浏览器本地存储（localStorage）或本地 JSON 文件，关闭网页不丢失

* 跨设备访问：同一 WiFi 环境下，电脑端打开本地网页后，手机 / 平板可通过输入电脑 IP 地址访问，数据实时同步（基于本地文件共享，无需服务器）

* 数据备份 / 恢复：支持导出数据为本地文件（如.json 格式），可手动导入备份文件恢复数据

## 三、界面与交互需求

### 1. 风格要求



* 整体风格：卡通活泼，配色以暖色系为主（如浅黄、浅粉、浅蓝），避免刺眼颜色

* 视觉元素：适当添加卡通食材图标（如青菜、肉、汤碗）、简约背景图案，提升趣味性

* 无多余元素：界面仅保留核心功能按钮，无广告、无复杂装饰

### 2. 交互设计



* 操作步骤：核心功能（如添加菜品、规划菜单）操作步骤≤3 步

* 按钮设计：功能按钮尺寸放大（手机端按钮宽度≥60px，电脑端≥80px），间距≥10px，避免误触

* 反馈清晰：点击按钮后有明显视觉反馈（如颜色变深、轻微动画），操作成功 / 失败有简洁提示（文字字号≥14px）

### 3. 适配要求



* 响应式布局：自动适配电脑、手机、平板屏幕，手机端优先竖屏展示，电脑端支持横屏

* 触控友好：支持手指点击、滑动操作（手机端），点击区域精准，无 “点不到” 情况

## 四、非功能需求

### 1. 性能要求



* 启动速度：本地网页打开时间≤3 秒，无加载延迟

* 操作流畅：点击、切换、搜索等操作无卡顿，响应时间≤1 秒

### 2. 数据安全



* 本地独占：数据仅存储在用户本地设备，不上传任何服务器，保护隐私

* 防丢失：支持手动备份数据，避免误删浏览器数据导致内容丢失

### 3. 易用性要求



* 无学习成本：无需说明书，家庭成员可通过界面指引快速上手

* 容错性：支持误操作撤回（如删除菜品后 3 秒内可点击 “撤销” 恢复）
