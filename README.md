# 家庭菜单小应用

一个简单易用的家庭菜单管理Web应用，帮助您轻松管理菜品库和菜单日历。

## 功能特性

### 1. 菜品库管理
- 添加、编辑、删除菜品
- 支持自定义分类
- 菜品搜索和分类过滤
- 按名称、分类、添加时间排序
- 直观的卡片式展示

### 2. 菜单日历
- 按月查看菜单
- 为每天的早餐、午餐、晚餐设置菜品
- 支持按餐次快速添加
- 随机添加菜品功能
- 菜品按分类组织展示

### 3. 数据管理
- 数据本地存储（localStorage）
- 数据导出功能
- 数据导入功能

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 在菜品库中添加您的菜品
3. 在菜单日历中为每天安排菜品
4. 随时可以导出/导入数据备份
